# Adminer Custom - Tri DESC par défaut

Adminer avec tri DESC par défaut sur les clés primaires.

## Usage

```bash
# Copier la config
cp .env.example .env

# Lancer
docker-compose up -d
```

Accès : http://localhost:8081

## Personnalisation

- Tri par défaut : DESC sur clé primaire
- Basé sur l'image officielle Adminer latest
